BEAST2_Tutorial

Ernesto

2023-12-13

Purpose

Learn the workflow of BEAST2 to interpret the result of phylogenetic analysis

Methods

The input data used comes from the sequence alignment of twelve primates mitochondrial genome.

We used HKY substitution model with empirical frequencies, the clock model was kept default

The priors model was Calibrated Yule Model with a birthrate set to gamma with alpha in 0.001 and Beta in 1000

The MCMC settings was set the chain length to 1,000,000 and 10,000,000 to compare the effects

Tracer allowed us to visualize the data in different ways

Tree Creation help us create the tree by annotating wit Tree annotator and modifying the aspect of it and fonts with FigTree

Results

The results was a phylogenetic tree with the data
